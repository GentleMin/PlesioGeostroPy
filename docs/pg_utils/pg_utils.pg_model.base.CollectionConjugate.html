class pg_utils.pg_model.base.CollectionConjugate(**fields)

Bases: LabeledCollection

Base class for the collection of conjugate variables, fields, equations, etc, which are the conjugate counterpart of PG variables, fields and equations.

__init__(**fields) → None

Initialization

Parameters:

  • names – list or array-like [str], list of names to be used as field attributes

  • **fields – keyword arguments to be initiated as attributes; the keys must be part of the names, otherwise the key-value pair is ignored; fields in names not in fields will be initiated as None; keys in fields not in names raises an error.
