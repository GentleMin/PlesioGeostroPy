class pg_utils.pg_model.expansion.FourierExpansions(argument: Expr, fields: LabeledCollection, fourier_coeffs: LabeledCollection)

Bases: SpectralExpansion

Fourier expansion for fields This class is used for reducing equations or expressions into the Fourier domain given specific arguments. The Fourier-domain eqs/ exprs are then processed using RadialExpansions

Parameters:

  • fields – fields to use the Fourier expansion

  • bases – the current basis would be the complex Fourier basis Note: unlike RadialExpansions, the Fourier basis is assumed to be shared among all fields. From another view, this basis has the same form in the expansion of different fields.

  • coeffs – Fourier coefficients

  • ansatz – the Fourier ansatz

__init__(argument: Expr, fields: LabeledCollection, fourier_coeffs: LabeledCollection) → None

Initiate Fourier expansion

Parameters:

  • arguments – the complex argument of the Fourier basis

  • fields – fields to use the Fourier expansion on

  • fourier_coeffs – Fourier coefficients; these should be Fourier coefficients of the fields specified by fields expanded with argument specified by argument.
