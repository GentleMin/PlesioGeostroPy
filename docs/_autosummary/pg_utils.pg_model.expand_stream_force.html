Expansion configuration file - Expansion for the streamfunction-force formulation

..Note: this set of bases and expansions has to be used

together with the reduced system of equations, where the only equations present are Psi and F_ext

..Note: this is only relevant to eigenvalue problems or

linearized systems. Nonlinear PG equations do not generally simplify into streamfunction - force formulation, but requires full set of PG variables or their conjugate counterparts
