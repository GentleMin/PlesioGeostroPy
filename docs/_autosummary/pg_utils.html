PlesioGeostroPy: a Python package for the Plesio-Geostrophy (PG) model.

This is the top-level module for PlesioGeostroPy, and includes four sub-modules:

  • pg_utils.sympy_supp (supplementary functions for symbolic calculation in sympy),

  • pg_utils.pg_model (realization of the symbolic PG model and spectral recipes),

  • pg_utils.numerics (associated numerics computations, quadrature + solvers etc.),

  • pg_utils.processing (pre- and post-processing).
